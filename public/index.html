About Panopt Lite

üéØ What is Panopt Lite?

Panopt Lite is a lightweight, self-hosted Security Information and Event Management (SIEM) system designed for individuals and small teams who want enterprise-grade security monitoring without the enterprise price tag.

üöÄ Key Features

  • Real-time Log Ingestion: Collect logs from multiple sources using Vector
  • AI-Powered Detection: Isolation Forest & Linear Regression for anomaly detection
  • Threat Intelligence: Automatic bad IP detection via AbuseIPDB
  • Beautiful Dashboards: Grafana-powered Mission Control interface
  • Multi-Platform Agents: Windows, Linux, Android, iOS support
  • Smart Alerting: Loki Rules + AlertManager with webhook notifications
  • Privacy First: 100% self-hosted, your data never leaves your network

üèóÔ∏è Architecture

Server Stack:

  • Grafana Loki: Log aggregation and storage
  • Grafana: Visualization and dashboards
  • Vector: High-performance log shipping
  • AlertManager: Alert routing and notifications
  • Python AI Engine: Anomaly detection and forecasting

üí° Use Cases

  • Home lab security monitoring
  • Small business infrastructure monitoring
  • Personal device fleet management
  • Learning SIEM concepts and security operations
  • Privacy-focused alternative to cloud SIEM solutions

How to Use Panopt Lite

1

Start the Server

On your central machine (Windows/Linux/Mac):

git clone https://github.com/your-repo/panopt-lite.git
cd panopt-lite
docker-compose up -d

Wait 30 seconds for all services to start.

2

Access Mission Control

Open your browser and navigate to:

http://localhost:3000

Login credentials:

  • Username: admin
  • Password: panopt_secure
3

Deploy Windows Agent

cd windows_agent
.\start-all.ps1

This starts both Vector (CPU/Memory/Disk) and Network collector.

4

Configure Alerts (Optional)

Enable Discord/Slack notifications:

  • Get webhook URL from Discord/Slack
  • Edit config/alertmanager/alertmanager.yml
  • Uncomment webhook section and add your URL
  • Restart: docker-compose restart alertmanager
5

Monitor & Enjoy!

Your Mission Control dashboard now shows:

  • ‚úÖ Memory consumption
  • ‚úÖ Network traffic (RX/TX)
  • ‚úÖ AI-detected anomalies
  • ‚úÖ Security alerts
  • ‚úÖ Real-time log stream
